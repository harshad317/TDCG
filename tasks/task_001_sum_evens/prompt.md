# Task: sum_evens

Implement the function `sum_evens(nums)` in `solution.py`.

## Spec
- Input: a list of integers (may be empty, may contain negatives).
- Output: an integer — the sum of all even numbers in the list.
- Zero is even.
- If the list is empty or has no even numbers, return 0.

## Example
```
sum_evens([1, 2, 3, 4]) == 6
sum_evens([])           == 0
sum_evens([-2, -1, 0])  == -2
```

A starter file `solution.py` is provided. Public tests are in `public_tests.py` — run them with `python -m pytest public_tests.py`.
